Overview

About the Simulator

The FG-8085 Simulator Toolkit is a comprehensive web-based simulator for the Intel 8085 microprocessor. It provides a user-friendly interface for writing, executing, and debugging 8085 assembly programs.

Shortcut Keys

Enter Go to next address
Shift + Enter Go to previous address

8085 Instruction Set

1. Data Transfer Instructions

Instruction Description Example
MVI r, data Move 8-bit immediate data to register MVI A, 32H
MOV r1, r2 Copy contents of register r2 to register r1 MOV A, B
LDA addr Load accumulator from memory address LDA 2050H
STA addr Store accumulator contents to memory address STA 2050H
LHLD addr Load H & L registers from memory address LHLD 2050H
SHLD addr Store H & L registers to memory address SHLD 2050H

2. Arithmetic Instructions

Instruction Description Example
ADD r Add register to accumulator ADD B
ADI data Add immediate data to accumulator ADI 32H
ADC r Add register to accumulator with carry ADC C
SUB r Subtract register from accumulator SUB B
SUI data Subtract immediate data from accumulator SUI 32H
SBB r Subtract register from accumulator with borrow SBB C
INR r Increment register by 1 INR B
DCR r Decrement register by 1 DCR B

3. Logical Instructions

Instruction Description Example
ANA r AND register with accumulator ANA B
ANI data AND immediate data with accumulator ANI 32H
ORA r OR register with accumulator ORA B
ORI data OR immediate data with accumulator ORI 32H
XRA r XOR register with accumulator XRA B
XRI data XOR immediate data with accumulator XRI 32H
CMA Complement accumulator CMA
CMC Complement carry flag CMC

4. Branch Instructions

Instruction Description Example
JMP addr Unconditional jump to address JMP 2050H
JZ addr Jump if zero flag is set JZ 2050H
JNZ addr Jump if zero flag is not set JNZ 2050H
JC addr Jump if carry flag is set JC 2050H
JNC addr Jump if carry flag is not set JNC 2050H
CALL addr Call subroutine at address CALL 2050H
RET Return from subroutine RET

5. Stack Instructions

Instruction Description Example
PUSH rp Push register pair onto stack PUSH B
POP rp Pop register pair from stack POP B
XTHL Exchange H-L with top of stack XTHL
SPHL Copy H-L register pair to stack pointer SPHL

6. I/O Instructions

Instruction Description Example
IN port Input from port to accumulator IN 01H
OUT port Output from accumulator to port OUT 01H

7. Machine Control Instructions

Instruction Description Example
HLT Halt program execution HLT
NOP No operation NOP
DI Disable interrupts DI
EI Enable interrupts EI
